# Usage and Limits

The **Usage and Limits** view provides comprehensive monitoring of your organization's **DataStream** resource consumption and subscription limits. This dashboard enables administrators to track data ingestion patterns, monitor resource utilization, and manage capacity planning across all organizational tenants.

---

To access the **Usage and Limits** view:

- Go to the **Home** > **Organization** pane
- Click **Usage and Limits**

-or-

- Click the hamburger menu on the top left
- Select **Organization** > **Usage and Limits**

The interface displays: "Viewing details for your own organization and all associated tenants."

## Usage Limit Exceeded Banner

When resource limits are approached or exceeded, a banner notification appears at the top of the page:

- **Warning Banner** - Prominent notification for limit violations
- **Usage Information** - Details about which resource exceeded limits
- **Action Guidance** - Links to upgrade or contact sales for limit increases
- **Dismissible Display** - Banner can be temporarily dismissed

## Tenant Filter

- **Filter Dropdown** - Select "All" or specific tenant to view (defaults to "All")
- **Multi-tenant Visibility** - Monitor usage across all associated tenant organizations
- **Aggregated View** - "All" option shows combined usage across all tenants
- **Tenant-Specific View** - Select individual tenant to view isolated usage data

## Resource Monitoring Cards

The left side displays your current resource utilization:

### Daily Data Ingestion Limit
- **Current Usage** - Real-time data consumption with unit conversion (e.g., "0 B of 5000 GB")
- **Progress Bar** - Visual indicator with colored fill showing consumption percentage
- **Usage Percentage** - Precise percentage display (e.g., "0% of limit used")
- **Capacity Planning** - "Need to upscale daily ingest limit?" with **Talk to sales** link
- **Always Visible** - Data ingestion limit applies to all subscription plans

### Directors

**Standard/Basic Plans:**
- **Current Count** - Number of active Director instances (e.g., "2 of 10")
- **Progress Bar** - Visual indicator showing allocation against limit
- **Remaining Count** - Shows available Directors (e.g., "8 directors remaining")

**Enterprise Plan:**
- **Current Count** - Number of active Director instances (count only)
- **Enterprise Label** - Displays "Enterprise" indicator
- **No Limits** - Progress bar hidden for unlimited Director deployments

### Agents

**Standard/Basic Plans:**
- **Current Count** - Number of active Agent deployments (e.g., "50 of 100")
- **Progress Bar** - Visual indicator showing allocation against limit
- **Remaining Count** - Shows available Agents (e.g., "50 agents remaining")

**Enterprise Plan:**
- **Current Count** - Number of active Agent instances (count only)
- **Enterprise Label** - Displays "Enterprise" indicator
- **No Limits** - Progress bar hidden for unlimited Agent deployments

## Data Analytics Dashboard

The right side provides detailed ingestion analytics:

### Ingested Data Amount

**Time Period Selection:**
- **Period Dropdown** - Select timeframe for data analysis (Last 7 days, Last 30 days, etc.)

**Usage Metrics:**
- **Daily Average** - Average data ingestion per day over selected period
- **Total Volume** - Cumulative data ingestion for the selected timeframe
- **Period Summary** - "Total of [X] B Last [Y] days"

### Analytics Display States

**Active Data Processing:**
- **Line Chart** - Time-series visualization of daily ingestion patterns
- **Data Points** - Daily breakdown showing consumption trends
- **Threshold Indicators** - Visual threshold lines on chart showing limit zones
- **Threshold Labels** - Positioned labels indicating:
  - **Close to Limit** - Warning zone approaching daily limit
  - **Daily Limit** - Maximum allowed ingestion threshold
- **Interactive Tooltips** - Hover over data points to view:
  - Formatted date and time with timezone
  - Exact data ingestion value with unit (e.g., "150.25 GB")
- **Threshold Highlights** - Color-coded background zones showing proximity to limits

**No Data Available:**
- **Empty State Icon** - Search icon displayed when no data exists
- **"No data found"** message when no ingestion activity exists
- **Information Note:** "There is no ingested data information available"
- Indicates new deployments or inactive periods

**Error State:**
- **Error Icon** - Variable icon displayed on fetch failure
- **Error Title** - Clear error message for data retrieval issues
- **Error Description** - Guidance for resolving data access problems

## Capacity Management

### Limit Monitoring
- Visual progress indicators for each resource type
- Percentage-based utilization tracking
- Early warning indicators for approaching limits

### Upgrade Options
- Direct sales team contact for limit increases
- Capacity planning guidance and recommendations
- Subscription modification pathways for growing organizations

This interface serves as the central resource management hub for **DataStream** administrators, providing real-time visibility into organizational data processing consumption and infrastructure utilization patterns.
