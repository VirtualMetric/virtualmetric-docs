## VirtualMetic Director

**VirtualMetic Director** is a tool designed for creating and managing _pipelines_ in tandem to monitor telemetry data streams for various purposes, and **VirtualMetic Generator** is a utility program intended to generate realistic telemetry data to help configure **Director** for production.

In this guide, we will explain the various features of the product: the types of devices, data files, and pipelines that can be created and specified; the data streams that can be handled; and the enrichment capabilities available in detail.
