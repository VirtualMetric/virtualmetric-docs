# Getting Started

## System Requirements

- **Memory**: 16MB or more to run **Director**
- **Disk**: ?GB or more for storing the log data
- **OS**: Windows 10, Windows 11, Linux, etc.

## Download

You can download **Director** by clicking [this link](www.example.com).

## Install

**VirtualMetric Director** does _not_ require any installation: you can place it in any directory you wish.

## Configure

You can start using **Director** without any initial configuration.

:::warning
Do not forget to put the directory path of **Director** in the environment `PATH` variable.
:::

## Run

Below, we refer to the directory where **Director** was placed as `<vm_root>`:

<Tabs>
  <TabItem value="windows" label="Windows" default>
  ```PowerShell
  C:\<vm_root>vmetric-director
  ```
  </TabItem>
  <TabItem value="linux" label="Linux">
  ```PowerShell
  ~/<vm_root>./vmetric-director
  ```
  </TabItem>
</Tabs>
