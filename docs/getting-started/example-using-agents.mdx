# Example: Using Agents

Configuration sample:

```yaml
devices:
    - id: 1
      name: localvpn
      type: windows
      status: true
      properties:
        debug:
          level: 4
          console:
            status: true
        type: agent
      definitions:
        - id: 491591344758837249  # Main
        - id: 486227243115741541
 
device:
  id: 2

director:
  address: nats://192.168.1.200:8080
```
