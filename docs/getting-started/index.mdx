# Getting Started: Overview

Welcome to your first installation and use attempt of **VirtualMetric DataStream**. In the following chapters, you can find the required procedures for deployment:

* **Single Node**: deploying on a [single machine](/docs/getting-started/single-node-deployment.mdx)

* **Cluster**: deploying on [multiple machines](/docs/getting-started/cluster-deployment.mdx) (possibly the cloud as well)

Once you complete the first deployment stage, we strongly advise you to see **Director** in action by trying the following preliminary examples in the indicated order:

1. [Example: A Syslog Device](/docs/getting-started/example-devices.mdx)

2. [Example: A Sentinel Target](/docs/getting-started/example-targets.mdx)

3. [Example: Routing Telemetry Data](/docs/getting-started/example-routes.mdx)

4. [Example: A Syslog To Sentinel Pipeline](/docs/getting-started/example-pipelines.mdx)
