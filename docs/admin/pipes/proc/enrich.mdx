# Enrich

:::info[synopsis]
Enriches documents with data from another index.
:::

|Field|Type|Required|Default|Description|
|---|---|---|---|---|
|`dataset_name`|Strings|Y|N/A||
|`lookup_table`|Strings|Y|N/A||
|`query`|String|Y|N/A||
|`description`|String|N|-|Explanatory note|
|`if`|String|N|-|Execute on this condition|
|`ignore_failure`|Logical|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`ignore_missing`|Logical|N|`false`|If set to `true` and `field` does not exist or is `null`, exit quietly without making any modifications|
|`on_failure`|Processors|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|Processors|N|||
|`override`|Logical|N|`true`|Update fields with pre-existing defined values. If set to `false`, such fields are not touched|
|`tag`|String|N|-|Identifier|
