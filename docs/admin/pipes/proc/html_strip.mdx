# HTML Strip

<Synopsis>
Removes **HTML** tags from a field, if the field is a string containing tags.
</Synopsis>

:::note
Each tag is replaced with a linebreak character.
:::

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`field`|Y|N/A|The field with the string containing **HTML** tags|
|`description`|N|-|Explanatory note|
|`if`|N|-|Execute on this condition|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`ignore_missing`|N|`false`|If `true` and `field` does not exist or is `null`, exit quietly without making any modifications|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|-|Use to inform the user that no errors were raised or to specify the next processor to run|
|`tag`|N|-|Identifier|
|`target_field`|N|`field`|The field to assign the string stripped off **HTML** tags|

## Examples

**Spec**:

```json
{
   "html_strip": {
      "field": "foo"
   }
}
```
