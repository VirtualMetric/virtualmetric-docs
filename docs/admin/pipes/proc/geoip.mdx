# Geo IP

<Synopsis>
Adds information about the geographical location of an IPv4 or IPv6 address.
</Synopsis>

## Parameters

|Field|Type|Required|Default|Description|
|---|---|---|---|---|
|`field`|String|Y|N/A|The field containing the IP address|
|`database_file`|String|N|?|Name of the database used to lookup the geographical information based on IP|
|`description`|String|N|-|Explanatory note|
|`first_only`|Strings|N|`true`|Return only the first ip geolocation info, disregarding others in `field` if any|
|`if`|String|N|-|Execute on this condition|
|`ignore_failure`|Logical|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`ignore_missing`|Logical|N|`false`|If `true` and `field` does not exist or is `null`, exit quietly without making any modifications|
|`on_failure`|Processors|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|String|N|-|Use to inform the user that no errors were raised or to specify the next processor to run|
|`properties`|Processors|N|[`continent_name`, `country_iso_code`, `country_name`, `region_iso_code`, `region_name`, `city_name`, `location`]|List of properties to be added to the `target_field`|
|`tag`|String|N|-|Identifier|
|`target_field`|String|N|-|The field that will store the geographical information|
