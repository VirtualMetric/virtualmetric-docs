# Sort

<Synopsis>
Sorts the elements of an array in ascending or descending order.
</Synopsis>

Arrays of numbers will be sorted numerically, whereas string arrays or heterogeneous arrays of strings and numbers will be sorted lexicographically.

Raises an exception when the field is not an array.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`field`|Y|N/A|The field to be sorted|
|`description`|N|-|Explanatory note|
|`if`|N|-|Condition to run|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|-|See [Handling Success](../handling-success.mdx)|
|`order`|N|`asc`|Sort order: Valid options: `asc`, `desc`|
|`tag`|N|-|Identifier|
|`target_field`|N|`field`|The field to assign the sorted values|

## Examples

**Spec**:

```js
{
   "sort": {
      "field": "foo",
      "order": "desc"
  }
}
```
