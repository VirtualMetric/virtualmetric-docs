# Gsub

:::info[synopsis]
Makes replacements in a string field using regular expressions. If the field is an array of strings, replacements will be applied to all members. If any non-string values exist, an exception will be raised.
:::

|Field|Type|Required|Default|Description|
|---|---|---|---|---|
|`field`|String|Y|N/A|The field to apply the replacement to|
|`pattern`|String|Y|N/A|The match pattern|
|`replacement`|String|Y|`field`|The replacement pattern|
|`description`|String|N|-|Explanatory note|
|`if`|String|N|-|Execute on this condition|
|`ignore_failure`|Logical|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`ignore_missing`|Logical|N|`false`|If set to `true` and `field` does not exist or is `null`, exit quietly without making any modifications|
|`on_failure`|Processors|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|Processors|N|-||
|`tag`|String|N-|Identifier|
|`target_field`|String|N|-|The field to assign the replacement-applied values to|

:::note[examples]
**Spec**:

```json
{
   "gsub": {
      "field": "id",
      "pattern": "\\-",
      "replacement": "_"
   }
}
```
:::
