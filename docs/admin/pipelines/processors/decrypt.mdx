# Decrypt

<Synopsis>
Decrypts a single encrypted string using a defined secret and a specified _Initialization Vector_ field.
</Synopsis>

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`algorithm`|Y|N/A|The algorithm used to encrypt the value in the field. Determinest the number of characters to use for the encryption key and the initialization vector|
|`encryption_key`|Y|N/A|The key used to encrypt the field|
|`field`|Y|N/A|Field to be decrypted|
|`iv_field`|Y|-|The value to be used by the algorithm as the initialization vector|
|`description`|N|-|Explanatory note|
|`if`|N|-|Condition to run|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`ignore_missing`|N|`false`|If `true` and `field` does not exist or is `null`, exit quietly without making any modifications|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|-|See [Handling Success](../handling-success.mdx)|
|`tag`|N|-|Identifier|
|`with_compression`|N|`false`|Flag to use compression|

## Examples

<Example>
   <Description>
   When the encrypted field _foo_
   </Description>
   <SampleCode>
   ```json
   {
      "iv_field": "FmXUb0OPOWm1A2kw6diKYw==",
      "foo": "vFza"
   }
   ```
   </SampleCode>

   <Description>
   is used with this algorithm
   </Description>
   <SampleCode>
   ```json
   AES-256-CFB (key=32 chars., iv=16 chars.)
   ```
  </SampleCode>

   <Description>
   ...and this encryption key
   </Description>
   <SampleCode>
   ```json
   6B58703273357638792F423F4528482B
   ```
   </SampleCode>

   <Description>
   ...it yields
   </Description>
   <SampleCode>
   ```json
   {
      "foo": "bar"
   }
   ```
   </SampleCode>
</Example>
