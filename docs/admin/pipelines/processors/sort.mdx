# Sort

## Synopsis

```yaml {2}
sort:
  - field: <ident>
  - description: <string>
  - if: <script>
  - ignore_failure: <logical>
  - on_failure: <processors...>
  - on_success: <processors...>
  - order: <enumerated>
  - tag: <string>
  - target_field: <ident>
```

## Description

**Functionality**: Sorts the elements of an array in ascending or descending order.

Arrays of numbers will be sorted numerically, whereas string arrays or heterogeneous arrays of strings and numbers will be sorted lexicographically.

Raises an exception when the field is not an array.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`field`|Y|N/A|Field to be sorted|
|`description`|N|-|Explanatory note|
|`if`|N|-|Condition to run|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|-|See [Handling Success](../handling-success.mdx)|
|`order`|N|`asc`|Sort order: Valid options: `asc`, `desc`|
|`tag`|N|-|Identifier|
|`target_field`|N|`field`|Field to assign the sorted values|

## Examples

Sort the values in the field in _descending_ order:

```yaml
- sort:
  field: foo
  order: desc
```
