# Final

## Synopsis

```yaml
fail:
  - description: <text>
  - if: <script>
  - ignore_failure: <boolean>
  - on_failure: <processors...>
  - on_success: <processors...>
  - tag: <string>
```

## Description

**Functionality**: Finalizes a pipeline without raising any errors.

Useful to prevent other pipelines from working on the same document.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`description`|N|-|Explanatory note|
|`if`|N|-|Condition to run|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|-|See [Handling Success](../handling-success.mdx)|
|`tag`|N|-|Identifier|

## Examples

Terminate the processor if it completes without errors:

```yaml
processor-1:
  field: foo
  target_field: bar
  on_success: final
```
