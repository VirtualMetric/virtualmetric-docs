# Append

<Synopsis>
Appends one or more values to an array.
</Synopsis>

If the field exists and is an array, the append operation is carried out. If the field exists and is a scalar, the field is converted to an array before the append operation. If the field doesn't exist, an array is created containing the provided values.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`field`|Y|N/A|Field to append the value to|
|`value`|Y|N/A|The value to be appended|
|`allow_duplicates`|N|`true`|If `false`, only distinct values are appended|
|`description`|N|-|Explanatory notes|
|`if`|N|-|Condition to run|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|-|See [Handling Success](../handling-success.mdx)|
|`tag`|N|-|Identifier|

## Examples

Append the values **bar** and the template value of the field **baz_id** to the **foo** field.

```yaml
append:
   - field: foo
   - value:
      - bar
      - {{{baz_id}}}
```
