# Encrypt

## Synopsis

```yaml {2-5}
encrypt:
  - algorithm: <enum>
  - encryption_key: <string>
  - field: <ident>
  - iv_field: <ident>
  - description: <text>
  - if: <script>
  - ignore_failure: <boolean>
  - ignore_missing: <boolean>
  - on_failure: <processors...>
  - on_success: <processors...>
  - tag: <string>
  - with_compression: <boolean>
```

## Description

**Functionality**: Applies an encryption algorithm to a field.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`algorithm`|Y||The algorithm to apply. Determines the number of characters to be used for the encryption key and the initialization vector|
|`encryption_key`|Y||The key to be used by the algorithm for the encryption|
|`field`|Y||The filed to be encrypted|
|`iv_field`|Y||Field used by the algorithm as the initialization key|
|`description`|N|-|Explanatory note|
|`if`|N|-|Condition to run|
|`ignore_failure`|N|`false`|See [Handling Failures](../handling-failures.mdx)|
|`ignore_missing`|N|`false`|If `true` and `field` does not exist or is `null`, exit quietly without making any modifications|
|`on_failure`|N|-|See [Handling Failures](../handling-failures.mdx)|
|`on_success`|N|||
|`tag`|N|-|Identifier|
|`with_compression`|N|||
