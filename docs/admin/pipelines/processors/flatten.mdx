# Flatten

<Synopsis>
Recursively reduces the level of objects, and appends the first level to their keys.
</Synopsis>

## Examples

<Example>
   <Description>
      Fields with multiple nested levels...
   </Description>
   <SampleCode>
      ```json
      {
         "foo": {
            "bar": {
               "baz": 10,
               "qux": "D"
            },
            "jaz": [1, 2, 3]
         }
      }
      ```
   </SampleCode>

   <Description>
      ...are flattened into a single level
   </Description>
   <SampleCode>
      ```json
      {
         "foo_bar_baz": 10,
         "foo_bar_qux": "D",
         "foo_jaz": [1, 2, 3]
      }
      ```
   </SampleCode>
</Example>
