# Flatten

## Synopsis

```yaml {2}
flatten:
  - fields: <idents...>
```

## Description

**Functionality**: Recursively reduces the level of objects, and appends the first level to their keys.

## Examples

<ExampleGrid>
  <CommentCol>
    Fields with multiple nested levels...
  </CommentCol>
  <CodeCol>
    ```json
    {
      "foo": {
        "bar": {
          "baz": 10,
          "qux": "D",
        },
        "jaz": [1, 2, 3]
      }
    }
    ```
  </CodeCol>

  <CommentCol>
    are flattened to a single level:
  </CommentCol>
  <CodeCol>
    ```yaml
    flatten:
      - fields:
        - foo.bar.baz
        - foo.bar.qux
        - foo.jaz
    ```
    ```json
    {
      "foo_bar_baz": 10,
      "foo_bar_qux": "D",
      "foo_jaz": [1, 2, 3]
    }
    ```
  </CodeCol>
</ExampleGrid>
