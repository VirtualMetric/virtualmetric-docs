# Flatten

## Synopsis

```yaml {2}
flatten:
   fields: <idents...>
```

## Description

**Functionality**: Recursively reduces the level of objects, and appends the first level to their keys.

## Examples

<Example>
   <Description>
      Fields with multiple nested levels...
   </Description>
   <SampleCode>
      ```yaml
      foo:
         - bar:
            - baz: 10
            - qux: "D"
         - jaz: [1, 2, 3]
      ```
   </SampleCode>

   <Description>
      ...are flattened into a single level:
   </Description>
   <SampleCode>
   ```yaml
   foo_bar_baz: 10
   foo_bar_qux: "D"
   foo_jaz: [1, 2, 3]
   ```
   </SampleCode>
</Example>
