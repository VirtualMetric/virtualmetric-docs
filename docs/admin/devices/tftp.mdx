# TFTP

## Synopsis

```yaml {5}
- id: <numeric>
  name: <ip-numeric>
  description: <text>
  type: tftp
  status: <logical>
  properties:
    address: <string>
    port: <numeric>
    device:
      <ip-numeric>: <port-numeric>
      ...
```

## Description

Represents the listener streaming data through the TFTP protocol.

## Parameters

<BasicConfig />

### `properties:`

Device-specific settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|Y|N/A|Listening port|

### `device:`

|Field|Required|Default|Description|
|---|---|---|---|
|`<ip-address>`|N|`<post-number>`||

## Examples

<details>
<summary>Sample code</summary>
```yaml
devices:
  - id: 386457759
    name: 127.0.0.1
    description: tftp
    type: tftp
    status: false
    properties:
      address: "0.0.0.0"
      port: 0
      device:
        127.0.0.1: 6942
        1.2.3.4: 5683
```
