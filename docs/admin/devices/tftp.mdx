# TFTP

## Synopsis

```yaml {5}
devices:
  - id: <numeric>
    name: <ip-numeric>
    description: <text>
    type: tftp
    status: <logical>
    properties:
      address: <string>
      port: <numeric>
      device:
        <ip-numeric>: <port-numeric>
        ...
```

## Description

Represents the listener streaming data through the TFTP protocol.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`id`|Y|N/A|Device identifier|
|`name`|Y|N/A|Address of the listener, e.g. `127.0.0.1`|
|`description`|N|-|Explanatory note|
|`type`|Y|N/A|Type of device. Should be `tcp`|
|`status`|N|`true`|Enable device|

### `properties:`

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|Y|N/A|Listening port|

### `device:`

|Field|Required|Default|Description|
|---|---|---|---|
|`<ip-address>`|N|`<post-number>`||
