# Redis

## Synopsis

```ssml {5}
- id: <numeric>
  name: <ip-numeric>
  description: <text>
  type: redis
  status: <logical>
  properties:
    address: <ip-string>
    port: <numeric>
    username: <string>
    password: <string>
    channel: <string>
    tls:
      status: <logical>
      cert_name: <string>
      key_name: <string>
```

## Description

Represents the listener streaming data from the Redis protocol.

## Parameters

<BasicConfig />

### `properties:`

Device-specific settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|N|`0`|Listening port|
|`username`|N|`""`|Required if authentication is used|
|`password`|N|`""`|Required if authentication is used|
|`channel`||`""`||

### `tls:`

Transport Layer Security settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`status`|N|`false`|Enable TLS connection|
|`cert_name`|N|`"cert.pem"`|TLS certificate file path|
|`key_name`||`"key.pem"`|TLS key file path|
