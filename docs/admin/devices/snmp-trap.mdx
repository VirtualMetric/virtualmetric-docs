# SNMP Trap

## Synopsis

```yaml
devices:
  - id: <numeric>
    name: <ip-numeric>
    description: <text>
    type: snmptrap
    status: <logical>
    properties:
      address: <ip-string>
      port: 0
      community: <string>
      username: <string>
      password: <string>
      channel: <string>
      authentication:
        status: <logical>
        protocol: <numeric>
        password: <string>
      privacy:
        protocol: <numeric>
        passphrase: <string>
```
