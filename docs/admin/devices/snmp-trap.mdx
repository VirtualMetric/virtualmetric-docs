# SNMP Trap

## Synopsis

```yaml {5}
devices:
  - id: <numeric>
    name: <ip-numeric>
    description: <text>
    type: snmptrap
    status: <logical>
    properties:
      address: <ip-string>
      port: <numeric>
      community: <string>
      username: <string>
      password: <string>
      authentication:
        status: <logical>
        protocol: <numeric>
        password: <string>
      privacy:
        protocol: <numeric>
        passphrase: <string>
```

## Parameters

<BasicConfig />

### `properties:`

Device-specific settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|Y|N/A|Listening port|
|`community`|N|`""`|Community string. Required if authentication v2 is used|
|`username`|N|`""`|Authentication username|
|`password`|N|`""`|Authentication password|

### `authentication:`

|Field|Required|Default|Description|
|---|---|---|---|
|`status`|N|`false`|Enable authentication v3. Required if authentication v3 is used|
|`protocol`|N|`1`|Authentication protocol. Required if authentication v3 is used|
|`password`|N|`""`|Authentication password. Required if authentication v3 is used|
