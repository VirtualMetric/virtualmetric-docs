# SMTP

## Synopsis

```ssml {5}
- id: <numeric>
  name: <ip-numeric>
  description: <text>
  type: smtp
  status: <logical>
  properties:
    address: <ip-string>
    port: <numeric>
    username: <string>
    password: <string>
    tls:
      status: <logical>
      cert_name: <string>
      key_name: <string>
    timeout: <numeric>
```

## Description

Represents the listener streaming data via the SMTP protocol.

## Parameters

<BasicConfig />

### `properties:`

Device-specific settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|Y|N/A|Listening port|
|`username`|N|`""`|Required if authentication is used|
|`password`|N|`""`|Required if authentication is used|
|`timeout`|N|`15`|Connection timeout in sec|

### `tls:`

Transport Layer Security settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`status`|N|`false`|Enable TLS connection|
|`cert_name`|N|`"cert.pem"`|TLS certificate file path|
|`key_name`|N|`"key.pem"`|TLS key file path|

## Examples

<Details summary="Sample Code">
```yaml
devices:
  - id: 324236543
    name: 127.0.0.1
    description: smtp
    type: smtp
    status: false
    properties:

      port: 0
      username: ""
      password: ""
      tls:
        status: false
        cert_name: ""
        key_name: ""
      reuse: false
      timeout: 15
```
