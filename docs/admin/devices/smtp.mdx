# SMTP

## Synopsis

```yaml {5}
devices:
  - id: <numeric>
    name: <ip-numeric>
    description: <text>
    type: smtp
    status: <logical>
    properties:
      address: <ip-string>
      port: <numeric>
      username: <string>
      password: <string>
      tls:
        status: <logical>
        cert_name: <string>
        key_name: <string>
      timeout: <numeric>
```

## Description

Represents the listener streaming data via the SMTP protocol.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`id`|Y|N/A|Device identifier|
|`name`|Y|N/A|Address of the listener, e.g. `127.0.0.1`|
|`description`|N|-|Explanatory note|
|`type`|Y|N/A|Type of device. Should be `smtp`|
|`status`|N|`true`|Enable device|

### `properties:`

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|Y|N/A|Listening port|
|`username`|N|`""`|Required if authentication is used|
|`password`|N|`""`|Required if authentication is used|
|`timeout`|N|`15`|Connection timeout in sec|

### `tls:`

|Field|Required|Default|Description|
|---|---|---|---|
|`status`|N|`false`|Enable TLS connection|
|`cert_name`|N|`"cert.pem"`|TLS certificate file path|
|`key_name`|N|`"key.pem"`|TLS key file path|
