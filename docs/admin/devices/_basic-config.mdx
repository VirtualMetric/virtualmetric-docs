Basic configuration.

|Field|Required|Default|Description|
|---|---|---|---|
|`id`|Y|N/A|Device identifier|
|`name`|Y|N/A|Address used to refer to the listener, e.g. `127.0.0.1`|
|`description`|N|-|Explanatory note|
|`type`|Y|N/A|Type of device, the value highlighted in the synopsis|
|`status`|N|`false`|Enable device|
