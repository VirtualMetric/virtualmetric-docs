Basic configuration.

|Field|Required|Default|Description|
|---|---|---|---|
|`id`|Y||Device identifier|
|`name`|Y||Address used to refer to the listener, e.g. `127.0.0.1`|
|`description`|N|-|Explanatory note|
|`type`|Y||Type of device, the value highlighted in the synopsis|
|`status`|N|`false`|Enable device|
