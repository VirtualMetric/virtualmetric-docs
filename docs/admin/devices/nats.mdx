# NATS

## Synopsis

```yaml
devices:
  - id: <numeric>
    name: <ip-numeric>
    description: <text>
    type: nats
    status: <logical>
    properties:
      address: <ip-string>
      port: 0
      username: <string>
      password: <string>
      stream: <string>
      consumer: <string>
      subject: <string>
      tls:
        status: <logical>
        cert_name: <string>
        key_name: <string>
```

## Description

Represents the NATS protocol.

NATS is a lightweight, high-performance messaging system that supports simple publish-subscribe and request-reply patterns. Known for its speed and simplicity, it is well-suited for microservices, IoT, and cloud-native architectures.
