# NATS

## Synopsis

```yaml {5}
devices:
  - id: <numeric>
    name: <ip-numeric>
    description: <text>
    type: nats
    status: <logical>
    properties:
      address: <ip-string>
      port: <numeric>
      username: <string>
      password: <string>
      stream: <string>
      consumer: <string>
      subject: <string>
      tls:
        status: <logical>
        cert_name: <string>
        key_name: <string>
```

## Description

Represents the listener that delivers the data collected with the NATS protocol.

NATS is a lightweight, high-performance messaging system that supports simple publish-subscribe and request-reply patterns. Known for its speed and simplicity, it is well-suited for microservices, IoT, and cloud-native architectures.

## Parameters

<BasicConfig />

### `properties:`

Device-specific settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`address`|N|`"0.0.0.0"`|Listening address|
|`port`|N|`0`|Listening port|
|`username`|N|`""`|Required if authentication is used|
|`password`|N|`""`|Required if authentication is used|
|`stream`|N|`"vmetric"`|Stream name|
|`consumer`|N|`"vmetric"`|Consumer name|
|`subject`|Y|N/A|Stream subject|

### `tls:`

|Field|Required|Default|Description|
|---|---|---|---|
|`status`|N|`false`|Enable TLS connection|
|`cert_name`|N|`"cert.pem"`|TLS certificcate file path|
|`key_name`|N|`"key.pem"`|TLS key file path|
