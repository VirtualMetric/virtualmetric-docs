# Azure Monitor

## Synopsis

```yaml
devices:
  - id: <numeric>
    name: <ip-address>
    description: <ident>
    type: azuremonitor
    status: <logical>
    properties:
      tenant_id: <string>
      client_id: <string>
      client_secret: <string>
      workspace_id: <string>
      stream:
        - <string>
        ...
```

## Description

Represents the listener that delivers data from **Azure Monitor**.

## Parameters

|Field|Required|Default|Description|
|---|---|---|---|
|`id`|Y|N/A|Device identifier|
|`name`|Y|N/A|Should be `127.0.0.1`|
|`description`|N|-|Explanatory note|
|`type`|Y|N/A|Type of device. Should be `azuremonitor`|
|`status`|N|-|Should be `false`?|
