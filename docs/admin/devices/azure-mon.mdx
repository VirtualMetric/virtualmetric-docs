# Azure Monitor

## Synopsis

```ssml {5}
- id: <numeric>
  name: <ip-address>
  description: <ident>
  type: azuremonitor
  status: <logical>
  properties:
    tenant_id: <string>
    client_id: <string>
    client_secret: <string>
    workspace_id: <string>
    stream:
      - <string>
      ...
```

## Description

Represents the listener delivering data from **Azure Monitor**.

## Parameters

<BasicConfig />

### `properties:`

Device-specific settings.

|Field|Required|Default|Description|
|---|---|---|---|
|`tenant_id`|Y|N/A|App registration tenant identifier|
|`client_id`|Y|N/A|App registration client identifier|
|`client_secret`|Y|N/A|App registration client secret key|
|`workspace_id`|Y|N/A|Log analytics workspace identifier|

### `stream:`

|Field|Required|Default|Description|
|---|---|---|---|
|&lt;`stream`&gt;|Y|N/A|Log analytics table name(s)|
