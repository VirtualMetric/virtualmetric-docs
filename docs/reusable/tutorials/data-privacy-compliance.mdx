:::warning Data Privacy & Compliance
When processing logs containing personal or sensitive information:

- **GDPR Compliance**: Ensure proper data handling for EU personal data
- **Data Minimization**: Only process and store necessary data fields
- **PII Handling**: Implement proper masking/redaction for personal information
- **Retention Policies**: Configure appropriate data retention and deletion
- **Audit Trails**: Maintain logs of data access and processing activities
- **Geographic Restrictions**: Consider data residency requirements
- **Consent Management**: Ensure proper consent for data processing where required

Review your organization's privacy policies and compliance requirements before deploying to production.
:::
