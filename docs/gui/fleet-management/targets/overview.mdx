# Targets: Overview

The Targets view displays target types in card format and provides detailed tables for each target type.

## Main View Interface

- **Search targets** text field at the top
- "Viewing X target type(s)" counter
- Target type cards displayed in a grid:

### Available Target Cards

- **Console**: Shows enabled/disabled counters with description
- **File**: Shows enabled/disabled counters with description
- **Microsoft Sentinel**: Shows enabled/disabled counters with description
- **Azure Blob Storage**: Shows enabled/disabled counters with description
- **Azure Data Explorer**: Shows enabled/disabled counters with description

## Target Type Table View

When selecting a target type (e.g., Microsoft Sentinel), the interface shows:

### Left Sidebar
- **Back to all targets** link
- Target type list with counts:
  - Console (with count)
  - File (with count)
  - Microsoft Sentinel (with count)
  - Azure Blob Storage (with count)
  - Azure Data Explorer (with count)

### Main Table Area
- **Search [Target Type] targets** field
- **Status** filter dropdown (showing "All")
- **Add new target** blue button

### Table Columns
- **Name**: Target identifier
- **Description**: Target description
- **Postprocessing Pipeline**: Associated pipeline
- **Status**: Green "Enabled" badges
- **Actions**: Three-dot menu

### Actions Menu
- **Edit target details**
- **Clone target**
- **Disable target**
- **Delete target**

### Bottom Controls
- **Items per page** dropdown
- "X-Y of Z items" display
- Page navigation showing current page
