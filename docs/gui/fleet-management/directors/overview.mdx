# Directors: Overview

The Directors view displays a table showing all configured directors with the following interface elements:

## Interface Elements

- **Search directors** text field at the top left
- **Status** dropdown filter (showing "All")
- **Create director** blue button at the top right
- **Refresh** button (circular arrow icon)

## Table Columns

The directors table displays:

- **Name**: Director identifier
- **Platform Type**: Shows "Self-managed" for all directors
- **Installation Type**: Shows "Standalone" for all directors
- **Status**: Green "Enabled" badges
- **Connection Status**: Shows "Connected" (green) or "Not Connected" (red)
- **Actions**: Three-dot menu button for each director

## Actions Menu

Clicking the three-dot menu reveals:

- **Edit director**
- **Disable director** 
- **Delete director**

## Bottom Controls

- **Items per page** dropdown (set to 10)
- Page navigation showing "1-10 of X items"
- Pagination controls with page numbers
