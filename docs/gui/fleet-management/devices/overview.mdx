# Devices: Overview

The Devices view shows device types in card format and provides detailed tables for each device type.

## Main View Interface

- **Search Devices** text field at the top
- "Viewing X device type(s)" counter
- Device type cards organized in two sections:

### Push Section

Displays cards for:
- **Syslog**: With enabled/disabled counters
- **HTTP**: With enabled/disabled counters  
- **UDP**: With enabled/disabled counters
- **TCP**: With enabled/disabled counters
- **eStreamer**: With enabled/disabled counters

### Pull Section

Displays cards for:
- **Windows**: With enabled/disabled counters
- **Linux**: Shows "Coming Soon" badge

## Device Type Table View

When selecting a device type (e.g., Syslog), the interface shows:

### Left Sidebar
- **Back to all devices** link
- Collapsible **Push devices** section with device counts
- Collapsible **Pull devices** section with device counts

### Main Table Area
- **Search devices** field
- **Directors** filter dropdown (showing "All")
- **Status** filter dropdown (showing "All")
- **Add new device** blue button

### Table Columns
- **Name**: Device identifier
- **Description**: Device description  
- **Director(s)**: Assigned directors
- **Preprocessing Pipeline**: Associated pipeline
- **Port**: Port number
- **Protocol**: Protocol type (UDP, TCP)
- **Status**: Green "Enabled" or "Disabled" badges
- **Actions**: Three-dot menu

### Actions Menu
- **Edit device details**
- **Clone device**
- **Disable device** / **Delete device**

### Bottom Controls
- **Items per page** dropdown
- "X-Y of Z items" display
- Page navigation controls
