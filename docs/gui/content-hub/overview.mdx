# Content Hub: Overview

The **Content Hub** displays available pipeline templates ready for use. The interface shows a search bar at the top, filtering options on the left sidebar, and template cards in the main area.

## Interface Layout

### Search and Description
- **Search Templates**: Text input field at the top for finding specific templates
- **Description text**: "Review available pipeline templates ready for use"
- **Template counter**: Shows "Viewing 20 templates" (or current count)

### Template Cards
Each template displays:
- **Template name**: Title of the pipeline template
- **Description**: Brief explanation of what the template does
- **Vendor logo**: Company or technology provider logo
- **Category tags**: Labels like "Syslog", "Microsoft Sentinel", "Firewall", etc.

## Left Sidebar Filters

The left sidebar contains collapsible filter sections:

### Device Type Filter
- **Windows (2)**: Shows templates for Windows systems
- **Wireless Access Point (1)**: Templates for wireless network devices
- **Cloud Security (3)**: Cloud security related templates
- **Syslog (3)**: Syslog message processing templates
- **Other (1)**: Miscellaneous device types
- **Load Balancer (1)**: Load balancer monitoring templates
- **Firewall (9)**: Firewall log processing templates

### Target Filter
- **Elasticsearch (3)**: Templates outputting to Elasticsearch
- **Other (1)**: Other target systems
- **Microsoft Sentinel (16)**: Templates for Microsoft Sentinel integration

### Device Vendor Filter
Expandable section showing vendor options with checkboxes for filtering templates by specific technology providers.

## Template Cards

The main area displays template cards in a grid layout. Each card shows:
- **Template title**: Name of the pipeline template
- **Description**: Brief explanation of functionality
- **Vendor logo**: Technology provider logo (Microsoft, Arista, Fortinet, etc.)
- **Category tags**: Colored labels indicating template type and target system

Click on any template card to view detailed information about that specific pipeline template.

## Bottom Section

At the bottom of the interface:
- **"Can't find what you are looking for?"** heading
- **Description text**: "Create your own pipeline or check VirtualMetric documentation for additional resources."
- **"Create new pipeline"** button with plus icon for building custom solutions
