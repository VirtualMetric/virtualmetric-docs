# Azure Sentinel Integration

The Content Hub provides pre-built Azure Sentinel integration templates for connecting DataStream with Microsoft's cloud-native SIEM solution.

## Integration Templates

The Content Hub includes templates for popular security platforms:

### Microsoft Security Integration
- **Microsoft 365 Defender**: Microsoft security ecosystem integration
- **Azure Security Center**: Cloud security posture management
- **Microsoft Defender for Endpoint**: Endpoint detection and response
- **Azure AD Integration**: Identity and access management logs

### Third-Party Security Platforms
- **Checkpoint Security**: Firewall and network security logs
- **Fortigate Integration**: Fortinet security appliance data
- **Cisco Security**: ASA, Meraki, and other Cisco security platforms
- **Palo Alto Networks**: Next-generation firewall integration

## Template Installation

To install Azure Sentinel integration templates:

1. Browse the Content Hub for available Azure Sentinel templates
2. Select the template that matches your security platform
3. Click **Install this pipeline template** to deploy the integration
4. Configure Azure workspace connectivity and authentication settings
