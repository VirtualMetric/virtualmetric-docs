# Creating Users

To access the **Users** view 

- Go to the **Home** > **Organization** pane
- Click **Manage Users**

-or-

- Click the hamburger menu on the top left
- Select **Organization** > **Users**

In the **Users** view

- Click **Add new user**.
- Enter the information for the user account to be created. See <Topic id="gui-user-details">**User Details**</Topic> for more information.

<Include id="user-creation-permissions" />

After entering the relevant details, click the **Add user** button to complete the process.
