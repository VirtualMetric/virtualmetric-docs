# Users: Overview

The **Users** page displays all registered users in your organization. The interface shows user information in a table format with search and filtering capabilities.

## Interface Elements

### Top Controls
- **Search users**: Text input field to find specific users by name or email
- **Status filter**: Dropdown menu with options "All", "Active", or "Passive" status
- **Add new user**: Blue button with "+" icon to create new user accounts

### User Table

The main table displays user information with these columns:
- **Full name**: User's complete name
- **Email**: User's email address
- **User Role**: Assigned role (Admin, Owner, User)
- **Status**: Current account status with colored indicators (green "Active" tags)

### Pagination Controls

- **Items per page**: Dropdown to select 10, 25, or 50 items per page
- **Page counter**: Shows current page range (e.g., "1-3 of 3 items")
- **Navigation**: Page numbers and navigation arrows (e.g. "1 of 1 pages")

Each row has a three-dot menu icon on the right for additional user actions.
