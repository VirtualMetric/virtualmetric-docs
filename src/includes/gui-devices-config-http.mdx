    #### Device Protocol Settings

    - **IP Address** (Optional) - IP address to be used for listening by the HTTP device. By default. `0.0.0.0`
    - **Port** - Listening port for the HTTP device. By default, `8080`

    <Include id="port-open-for-inbound-traffic" />

    <Include id="port-open-for-inbound-traffic" />

    - **URL** (Optional) - URL path the HTTP device will listen on. By default, `/`
    - **Content Type** (Optional) - MIME classification of the incoming requests. By default, `application/json`

    #### Response

    - **Code** (Optional) - HTTP response status code
    - **Body** (Optional) - HTTP response body content. By default, `{"message":"success"}`
    - **Content Type** (Optional) - MIME classification of the response content. By default, `application/json`

    #### Authentication

    - **Type** - Authentication type used by the HTTP device

    #### TLS Settings

    - **TLS Entryption** - Toggle to enable/disable TLS encryption on the HTTP device

    Click **Back** to return to the previous step, click **Next step** to continue.
  </TabItem>

  <TabItem value="advanced" label="Advanced Configuration">
    #### Performance

    - **Socket address reuse** - Toggle to enable/disable socket address reuse. By default, enabled
    - **Workers** (Optional) - Number of worker processes to spawn when reuse is enabled. By default, `0` which represents maximum number of workers

    #### Messages

    - **Queue Interval** (Optional) - Queue processing interval in seconds

  </TabItem>
