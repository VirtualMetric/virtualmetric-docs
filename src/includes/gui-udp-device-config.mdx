    #### Protocol Settings

    - **IP Address** (Optional) - IP address the UDP device will use to listen. By default, `0.0.0.0`
    - **Port** - Listening port for the UDP device. By default, `6004`

  </TabItem>
  <TabItem value="advanced" label="Advanced Configuration" default>
    #### Performance

    - **Socket address reuse** - Toggle to enable/disable socket reuse. By default, enabled
    - **Workers** (Optional) - Number of worker processes to spawn when reuse is enabled. By default, `0` which represent maximum workers
    - **Buffer Size** (Optional) - Network read buffer size in bytes. By default, `9000`
    
    #### Messages

    - **Batch Size** (Optional) - Number of messages to batch before flushing. By default, `1000`
    - **Queue Interval** (Optional) - Queue processing interval in seconds. By default, `1`
  </TabItem>
