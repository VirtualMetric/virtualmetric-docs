<Tabs>
  <TabItem value="file-properties" label="File Properties">
    - **Location** (Optional) - File output directory
    - **File name** (Optional) - File name template. By default, `vmetric.{{.Timestamp}}.{{.Extension}}` 
    - **Type** (Optional) - File format. By default, `json`
    - **Compression** (Optional) - Compression algorithm. By default, `zstd`
    - **Schema** - Data schema for Avro/OCF/Parquet formats
    - **No Buffer** - Toggle to enable/disable buffering. By default, disabled
    - **Format** - Field normalization format (ECS, CIM, ASIM, LEEF, etc.)
  </TabItem>
</Tabs>
